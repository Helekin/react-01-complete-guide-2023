## Getting Started

### Usage

1. Install dependencies

```sh
cd 20-easy-advance-nextjs
npm install
```

2. Rename the `.env.example` file to `.env.local` and add the following

```sh
MONGO_URI="your_mongo_uri"
```

3. Run the application in development mode

```sh
npm run dev
```
